# pre-commit

GitHub Action that runs pre-commit hooks.

## Usage

```yaml
    steps:
      - name: Checkout
        uses: actions/checkout@<commit-hash>
      - name: Run pre-commit
        uses: nikaro/actions/pre-commit@<commit-hash>
```
