# getsops

GitHub Action that install [SOPS](https://github.com/getsops/sops/).

## Usage

```yaml
    steps:
      - name: Checkout
        uses: actions/checkout@<commit-hash>
      - name: Run pre-commit
        uses: nikaro/actions/getsops@<commit-hash>
```
